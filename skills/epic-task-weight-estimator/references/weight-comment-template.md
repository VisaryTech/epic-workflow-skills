# Weight comment template

Use this template for every task comment after writing `task.weight`.

```text
Оценка веса: <hours>ч.
Основание:
- <factor 1>
- <factor 2>
- <factor 3>
```

## Rules

- The first line must always contain the final weight in hours.
- The `Основание:` block is mandatory.
- Include 2-4 short factors that materially influenced the estimate.
- Keep wording short and specific.
- Do not add vague filler or generic prose.
